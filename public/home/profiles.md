# Profiles

Build profiles select userspace contents.

Minimal boots the smallest headless system.

Standard is more than minimal: compiler and tooling support, network tools,
filesystem tools, and practical system utilities.

Desktop layers on the graphical system: Wayland, audio, WiFi, greetd,
Alpenglowed, foot, and browser shell pieces.

Kernel profiles are separate. Fast is the smallest boot path. Minimal adds the
networked appliance kernel surface. Desktop adds display, audio, USB, HID, WiFi,
Bluetooth, firmware, and desktop filesystems.
