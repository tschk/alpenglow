# Packages

Oil manages APK payloads. Recipes are in `system/oil/recipes/`.

```sh
oil search fastfetch
oil info fastfetch
oil install fastfetch
oil update
```

vro (vi) is included for editing.

```sh
vro README.md
```

`oil search` prints nothing when no package matches. `oil upgrade`/`outdated` print only when something needs action.
